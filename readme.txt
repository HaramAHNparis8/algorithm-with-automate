I implemented a finite state machine (automaton) using regular expressions. I considered efficient methods for processing data. I analyzed patterns using hashcodes, and to ensure the accuracy of the extracted data, I implemented and verified it using regex.
And in the remaining folders, you can find the practice codes I worked on for algorithms related to data management.

J'ai implémenté une machine à états finis (automate) en utilisant des expressions régulières. J'ai réfléchi à des méthodes efficaces pour traiter les données. J'ai analysé les motifs à l'aide de hashcodes, et pour garantir l'exactitude des données extraites, je les ai implémentées et vérifiées en utilisant des regex.
Et dans les dossiers restants, vous pouvez trouver les codes d'entraînement que j'ai réalisés pour les algorithmes liés à la gestion des données.


정규 표현식을 통해서  유한 상태 기계(automate)를 구현했습니다. 데이터를 어떻게 처리할지 효율적인 방법이 뭔지 생각해봤습니다. hashcode를 통해 패턴 분석을 해봤으며 뽑은 데이터가 정확한지 확인을 위해서 regex를 구현해 확인해봤습니다.그리고 나머지 폴더에는 데이터 관리를 위한 알고리즘을 위해 연습한 코드를 볼 수 있습니다.
